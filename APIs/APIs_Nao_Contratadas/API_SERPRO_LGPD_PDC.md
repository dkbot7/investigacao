# API/Plataforma SERPRO - LGPD / Privacidade Digital do Cidadão (PDC)

**Solução para Adequação à Lei Geral de Proteção de Dados Pessoais**

---

## VISÃO GERAL

O **SERPRO LGPD** é uma plataforma completa que oferece serviços e tecnologias para atender aos princípios da Lei Geral de Proteção de Dados Pessoais (Lei 13.709/2018).

A solução principal é a **Privacidade Digital do Cidadão (PDC)** - uma ferramenta para gestão de consentimentos e requisições de privacidade.

### Avaliação do TCU (2025)

O SERPRO obteve **91,47%** no indicador de adequação à LGPD, ocupando o **8º lugar** entre 387 organizações públicas federais avaliadas, com classificação **"Aprimorado"** (nível máximo).

## PRIVACIDADE DIGITAL DO CIDADÃO (PDC)

### Funcionalidades Gratuitas

| Funcionalidade | Descrição |
|----------------|-----------|
| **Portal do Cidadão** | Plataforma para exercício de direitos LGPD |
| **Requisições de Privacidade** | Canal para solicitações de titulares |
| **Sem limite de uso** | Funcionalidades básicas ilimitadas |

**Portal:** https://cidadao.pdc.serpro.gov.br/cidadao/

### Funcionalidades Pagas

| Funcionalidade | Preço |
|----------------|-------|
| **Solicitação de Consentimento** | R$ 0,08 a R$ 0,18 por transação |
| **Fluxo de Requisição de Privacidade** | R$ 0,08 a R$ 0,18 por transação |
| **Identificação do Titular** | Incluso no pacote |
| **Gestão de Consentimentos** | Incluso no pacote |
| **Canal de Relacionamento** | Para reclamações de privacidade |

### Gestão de Consentimentos

O PDC faz:
- **Envio automático** de solicitações de consentimento
- **Notificação** do cidadão por e-mail ou app MeuGov.br
- **Dashboard** com status das solicitações:
  - Autorizadas
  - Pendentes
  - Revogadas
  - Expiradas
- **Revogação online** sem interação com a empresa

## DIREITOS LGPD ATENDIDOS

Conforme Arts. 9º, 18 e 20 da LGPD:

| Artigo | Direito |
|--------|---------|
| **Art. 9º** | Acesso facilitado às informações |
| **Art. 18** | Direitos do titular (confirmação, acesso, correção, anonimização, portabilidade, eliminação, etc.) |
| **Art. 20** | Revisão de decisões automatizadas |

### Prazos de Atendimento

- **Até 15 dias** para atendimento às requisições

## CONFORMIDADE LGPD DAS APIs SERPRO

### API Consulta CPF

Os dados tratados são dados pessoais do CPF, utilizados para consulta e validação com bases de dados públicas preexistentes. Disponíveis também na página pública da SRFB.

**Aviso de Privacidade:** https://www.serpro.gov.br/privacidade-protecao-dados/aviso-de-privacidade-api-cpf

### API Consulta CNPJ

Os dados tratados são dados pessoais do CNPJ (incluindo sócios), utilizados para consulta e validação com bases de dados públicas preexistentes.

**Aviso de Privacidade:** https://www.serpro.gov.br/privacidade-protecao-dados/aviso-de-privacidade-api-cnpj

## CASOS DE USO

### 1. Gestão de Consentimentos
Empresas gerenciando autorizações de clientes para uso de dados.

### 2. Portal de Privacidade
Canal para titulares exercerem direitos LGPD.

### 3. Auditoria e Compliance
Registros de todas as operações com dados pessoais.

### 4. Identificação Segura
Uso de biometria para confirmar identidade do titular.

### 5. Notificação de Titulares
Envio automatizado de solicitações de consentimento.

### 6. Revogação de Consentimento
Permitir que titulares revoguem autorizações online.
