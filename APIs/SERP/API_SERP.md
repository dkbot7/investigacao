# SERP - Sistema Eletrônico de Registros Públicos

**Plataforma Nacional de Consulta de Registros Civis**

---

## VISÃO GERAL

O SERP é a plataforma eletrônica oficial dos Registros Públicos do Brasil, estabelecida pela **Lei n. 14.382/2022**. Permite consulta e solicitação de certidões de nascimento, casamento e óbito em todo o território nacional.

| Item | Valor |
|------|-------|
| **Site Principal** | https://serp.registros.org.br |
| **SERP-Jud** | https://serph.registros.org.br |
| **Portal Cidadão** | https://registrocivil.org.br |
| **Base Legal** | Lei 14.382/2022 |
| **Operador** | ONSERP (Operador Nacional do SERP) |
| **Regulação** | CNJ - Conselho Nacional de Justiça |

---

## SERVIÇOS DISPONÍVEIS

### 1. Consulta de Registros

| Tipo de Registro | Disponível |
|------------------|------------|
| Certidão de Nascimento | ✅ Sim |
| Certidão de Casamento | ✅ Sim |
| **Certidão de Óbito** | ✅ Sim |
| Registro de Imóveis | ✅ Sim |
| Títulos e Documentos | ✅ Sim |

### 2. SERP-Jud (Para Poder Judiciário)

Plataforma que permite consulta unificada em todo território nacional:

| Funcionalidade | Descrição |
|----------------|-----------|
| **Certidões RC** | Nascimento, casamento e óbito |
| **Pesquisa de Bens** | Nacional |
| **Matrículas Imóveis** | Visualização |
| **CNG** | Central Nacional de Garantias |

---

## DADOS PARA CONSULTA

| Dado | Obrigatório | Descrição |
|------|-------------|-----------|
| **CPF** | Recomendado | Busca mais precisa |
| **Nome Completo** | Sim | Nome da pessoa |
| **Data de Nascimento** | Não | Ajuda na busca |
| **Data de Óbito** | Não | Se conhecida |
| **Nome da Mãe/Pai** | Não | Refina resultados |

---

## O QUE RETORNA (ÓBITO)

| Campo | Descrição |
|-------|-----------|
| **Cartório** | Nome do cartório que registrou |
| **Cidade/UF** | Localização do cartório |
| **Livro/Folha/Termo** | Dados do registro |
| **Data do Registro** | Quando foi registrado |
| **Matrícula** | Número da matrícula (se disponível) |

---

## FORMAS DE ACESSO

### 1. Portal do Cidadão (Gratuito)
```
https://registrocivil.org.br
```
- Busca de registros
- Solicitação de 2ª via
- Entrega em casa, email ou cartório

### 2. SERP-Jud (Autenticação)
```
https://serph.registros.org.br
```
Autenticação:
- Certificado Digital ICP-Brasil
- IdRC (Sistema de Autenticação do Registro Civil)

### 3. PDPJ-Br
Acesso via Plataforma Digital do Poder Judiciário Brasileiro

## CUSTO

| Serviço | Custo |
|---------|-------|
